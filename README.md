# ErKai-开源项目二开计划

本项目计划寻找100个优秀的开源安全项目代码深入分析学习后进行二次开发。项目包括包括优秀的安全工具代码或优秀的安全项目框架等。希望通过二开计划，能够站在优秀开发者的肩膀上看Golang！

本项目创建于2021年3月7日，最近的一次更新时间为2022年11月02日。本项目会持续更新，直到海枯石烂！
